# Introduction to LAN

## Overview

This room introduced the fundamentals of computer networking and how devices communicate within a Local Area Network (LAN).

## Topics Covered

- Local Area Networks (LAN)
- Networking Devices
- OSI Model
- Packets
- Frames

## Key Learnings

### Local Area Network (LAN)
A LAN connects devices within a limited geographical area, allowing them to communicate and share resources efficiently.

### OSI Model
Learned about the seven layers of the OSI model and how each layer contributes to data communication.

### Packets
Data is divided into packets before being transmitted across a network.

### Frames
Packets are encapsulated into frames for transmission over the physical network.

## Reflection

Understanding networking fundamentals is essential before exploring penetration testing and network security.
