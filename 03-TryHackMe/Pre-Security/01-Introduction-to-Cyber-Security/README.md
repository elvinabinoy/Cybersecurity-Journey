# Introduction to Cyber Security

## Overview

This room introduced the fundamentals of cybersecurity, the different areas within the field, and the career opportunities available.

## Topics Covered

- What is Cybersecurity?
- Offensive Security
- Defensive Security
- Common Cybersecurity Roles
- Importance of Cybersecurity

## Key Learnings

### Offensive Security
Focuses on identifying vulnerabilities by simulating real-world attacks. Professionals in this field help organizations strengthen their security by finding weaknesses before attackers do.

### Defensive Security
Focuses on protecting systems, monitoring networks, detecting threats, and responding to security incidents to keep an organization's infrastructure secure.

### Career Paths
- Security Analyst
- Penetration Tester
- Security Engineer
- Incident Responder
- Cloud Security Engineer
- Digital Forensics Analyst
